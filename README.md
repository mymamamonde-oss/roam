# Roam — by My Mama Monde

A family travel planner. This folder is a ready-to-deploy website.

## What's inside
- `src/App.jsx` — the whole app
- `public/hero.jpg` — your hero photo (already wired in)
- everything else — the standard setup a website needs

## To change your hero photo later
Replace `public/hero.jpg` with a new photo of the same name. Done.

## To update your Amazon affiliate tags
Open `src/App.jsx`, find the `AFFILIATE` block near the very top, paste your tags.

## Deploy
See the step-by-step in the chat. Short version:
1. Put this folder on GitHub
2. Import it into Vercel
3. Vercel auto-detects Vite and deploys it
